<h1 align="center">
babel-plugin-dynamic-module
</h1>

<div align="center">
支持动态模块的 Babel 插件
</div> 

## 🏗 安装

```bash
# npm install
npm install @sensoro/babel-plugin-dynamic-module --save --dev

# yarn install
yarn add --dev @sensoro/babel-plugin-dynamic-module

# pnpm install
pnpm i --dev @sensoro/babel-plugin-dynamic-module
```
